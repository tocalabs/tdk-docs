# Boolean

## Defintion

The `Boolean` type in Toca is just the [boolean](https://docs.microsoft.com/en-us/dotnet/api/system.boolean?view=net-6.0) type in C#.

## Methods
There are no additional methods implemented for the `Boolean` type but you are able to use it as a normal `Boolean`.
```csharp
let failOnError = true;
if (failOnError) 
{
    AddDebugLog($"Fail on Error flag set to true");
}
```
